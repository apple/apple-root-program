# Apple Root Program

This repository contains the official policy documentation and resources for the Apple Root Program.

## About

The Apple Root Program establishes requirements for Certification Authorities (CAs) whose root certificates are included in Apple operating systems and devices. This includes requirements for certificate issuance, security practices, and operational standards.
